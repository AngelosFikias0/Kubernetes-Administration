# kubernetes-administration

Notes, manifests, and lab solutions covering core Kubernetes administration.
Based on the KodeKloud curriculum by Mumshad Mannambeth.

## Coverage

- Core Concepts (Pods, ReplicaSets, Deployments, Services, Namespaces)
- Scheduling (Taints, Affinity, DaemonSets, Static Pods, Priority Classes)
- Logging & Monitoring
- Application Lifecycle (Rolling Updates, ConfigMaps, Secrets, Autoscaling)
- Cluster Maintenance (Upgrades, Backup & Restore, ETCD)
- Security (TLS, RBAC, Network Policies, Service Accounts)
- Storage (PV, PVC, Storage Classes, CSI)
- Networking (CNI, DNS, Ingress, Gateway API)
- Helm & Kustomize
- Troubleshooting
